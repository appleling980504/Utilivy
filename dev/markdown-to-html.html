Markdown Cheatsheet Markdown 速查表

  • ▸ Headers: # H1, ## H2, ### H3
  • ▸ 标题: # H1, ## H2, ### H3
  • ▸ Bold: **bold** or __bold__
  • ▸ 粗体: **粗体** 或 __粗体__
  • ▸ Italic: *italic* or _italic_
  • ▸ 斜体: *斜体* 或 _斜体_
  • ▸ Links: [text](url)
  • ▸ 链接: [文本](网址)
