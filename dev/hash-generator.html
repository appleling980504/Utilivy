About Hashes 关于哈希

  • ▸ Fixed-size output from variable input
  • ▸ 可变输入生成固定大小输出
  • ▸ Used for data integrity verification
  • ▸ 用于数据完整性验证
  • ▸ One-way function (cannot be reversed)
  • ▸ 单向函数（无法反转）
  • ▸ Small changes create completely different hashes
  • ▸ 微小变化会产生完全不同的哈希
