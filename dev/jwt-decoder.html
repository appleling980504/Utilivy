About JWT 关于 JWT

  • ▸ JSON Web Tokens for secure data transfer
  • ▸ 用于安全数据传输的 JSON Web Tokens
  • ▸ Consists of Header, Payload, and Signature
  • ▸ 由头部、负载和签名组成
  • ▸ Commonly used for authentication
  • ▸ 常用于身份验证
  • ▸ Base64Url encoded JSON data
  • ▸ Base64Url 编码的 JSON 数据
