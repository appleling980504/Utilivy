How to Use 使用说明

  1. 1. Enter text in the input field
  2. 1. 在输入框中输入文本
  3. 2. Select the desired case conversion
  4. 2. 选择需要的大小写转换方式

  6. 3. 在结果区域查看转换后的文本

Conversion Types 转换类型

  • ▸ UPPERCASE: Converts all characters to uppercase
  • ▸ 大写: 将所有字符转换为大写
  • ▸ lowercase: Converts all characters to lowercase
  • ▸ 小写: 将所有字符转换为小写
  • ▸ Title Case: Capitalizes the first letter of each word
  • ▸ 标题格式: 每个单词的首字母大写
  • ▸ Sentence case: Capitalizes the first letter of each sentence
  • ▸ 句子格式: 每个句子的首字母大写
